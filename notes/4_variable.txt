
    # What is variable ?
    variable เป็นชื่อที่เราตั้งให้กับ data
    ใช้สำหรับจัดเก็บข้อมูลชั่วคราว

    # Naming rules
        - Case sensitive
        - สามารถใช้ ตัวอักษร, ตัวเลข. _, $ ในการตั้งชื่อ
        - เริ่มต้นด้วย ตัวอักษร, _, $
        - ต้องไม่ใช่ keyword
        - ต้องไม่ซ้ำกับ class name
        - ตั้งชื่อยาวได้ไม่จำกัดความยาว
        - ใช้ CamelCase
